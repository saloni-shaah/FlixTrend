# **App Name**: TrendStream

## Core Features:

- Personalized News Feed: Display posts from followed users and recommended content in a dynamically updating feed.
- User Profile Customization: Allow users to customize their profiles with personal information, profile pictures, and cover photos.
- Direct Messaging: Enable users to send private messages to their friends and other users.
- Content Posting: Allow users to create and post text updates, photos, and videos to their news feed and profile.
- Trending Topics Discovery: AI powered tool to find trending topics. Generate a list of trending topics to be displayed in its own section.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to establish a friendly and inviting atmosphere, evoking trust and approachability.
- Background color: Very light blue (#F0F8FF) to offer a clean and unobtrusive backdrop, improving readability and focusing attention on content.
- Accent color: Vibrant purple (#A78BFA) used sparingly to highlight key interactive elements and CTAs, adding a modern touch and visual interest.
- Body and headline font: 'PT Sans' for a clean and readable user interface.
- Use minimalist and consistent icons for navigation and actions.
- Implement a clean, grid-based layout for a visually organized and intuitive user experience.
- Use subtle transitions and animations for feedback and to enhance the user experience.
